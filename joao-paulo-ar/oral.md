# Prova oral N1 — João Paulo Alves da Rocha

Perguntas para a explanação de 5 minutos. Respostas orais, sem consultar o artigo no momento.

1. Vocês definem agregação à moda de Chen (MER estendido, "compra" vira entidade), mas em UML agregação é "tem um" e composição é "parte de". Esses dois sentidos são equivalentes? Qual cabe melhor para modelar uma fábrica?

2. Você cita o CODASYL nos anos 60. O que era o CODASYL e que limitações dos modelos hierárquico e em rede o modelo relacional de Codd (1970) veio resolver?

3. Você expande ACID e BASE corretamente. Como o teorema CAP justifica a coexistência dos dois? Qual letra do CAP cada modelo prioriza?

4. A seção 2.2 trata MVC e camadas. Inclua a arquitetura hexagonal (Cockburn) e a Clean Architecture (Martin): como cada uma trata o banco, e em qual ele é mais explicitamente detalhe de infraestrutura?

5. A seção 2.7 cita o SQL Server como SGBD. Descreva o procedimento de conexão a um banco relacional: o que precisa na string de conexão, qual biblioteca Python e como validar que a extração trouxe dados íntegros. Depois, percorra o ciclo dado → informação → conhecimento → decisão usando uma consulta SQL genérica de produção.
